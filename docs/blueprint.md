# **App Name**: StudySpark

## Core Features:

- Public Landing Page: A multi-section landing page with a header, hero section, about section, features showcase, public search, feedback form, and footer, designed to attract new users and provide key information about StudySpark.
- Authentication Modal: Interactive 'flip card' modal for user login and signup, integrated with Firebase Auth, redirecting users to the dashboard upon successful authentication.
- Personalized Dashboard: Provides a high-level overview of the user's academic status, displaying key metrics, weekly activity, upcoming assignments, and a grade estimator.
- Subjects Management: Enables users to add, edit, and manage their subjects, with each subject card displaying the subject title, instructor, and progress bar. Offers detailed views including assignments table, grade calculator, and study materials.
- Interactive Calendar: Highlights dates with due assignments, allowing users to add, edit, or delete assignments directly from the calendar view, thus aiding in deadline management.
- AI Tutor: AI tool that takes questions about course material and provides structured responses.
- AI Summarizer: Summarizes larger pieces of texts using generative AI

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke a sense of focus and intellect.
- Background color: Very light lavender (#F0F8FF) to provide a calm and uncluttered backdrop.
- Accent color: Vivid Purple (#9400D3) to highlight interactive elements and calls to action.
- Body font: 'Inter', a sans-serif font, for its modern and neutral appearance, suitable for both headlines and body text. 
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent and clear icons from Lucide to represent different features and actions, enhancing usability.
- Implement a responsive and clean layout, leveraging shadcn/ui components to maintain consistency and readability across devices.
